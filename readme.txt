Connects WooCommerce to Trexle giving the ability to accept online payments from 100+ payment gateways like Stripe, Authorize.net, and Braintree.

== Description ==

Connect WooCommerce to Trexle so you can use +100 payments gateways.

**What is Trexle?**

*"Trexle is an online billing solution that gives the ability to use 100+ payment gateways to accept online payments. Trexle integrates with dozens of shopping carts and e-commerce platforms."*

Trexle WooCommerce extension allows any Trexle active customer to accept online payments on his WooCommerce enabled site.


**[Check out the full features list](https://trexle.com)**


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wootrexle` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the WooCommerce settings screen to add your Trexle API keys.
